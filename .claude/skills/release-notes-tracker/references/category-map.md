# Карта категорій

Що куди категоризувати при обробці release notes.

| Категорія | Файл | Що сюди входить |
|-----------|------|---------------|
| Основні | `core.md` | Thinking, зусилля, план-режим, fast mode, моделі, контекст/пам'ять, інтерактивність, /compact, /export, /copy, /stats, /status, /doctor, /debug, /color, /timer |
| CLI | `cli.md` | Нові CLI прапорці, `claude` підкоманди, `claude mcp`, `claude auth`, `claude plugin`, `claude agents`, `claude remote-control` |
| Налаштування | `settings.md` | Ключі settings.json, дозволи, контекст, MCP, sandbox, Managed/Enterprise |
| Змінні середовища | `env-vars.md` | Змінні (API, таймаути, поведінка, шляхи, телеметрія) |
| Клавіатурні скорочення | `keybindings.md` | Навігація, vim mode, голос, кастомні прив'язки |
| MCP | `mcp.md` | Транспорти, OAuth, пошук інструментів, connectors, channels |
| Хуки | `hooks.md` | Типи подій, налаштування, вихідні дані |
| Навички та Агенти | `skills-agents.md` | Skills, агенти, slash команди, rules, frontmatter поля |
| Плагіни | `plugins.md` | Джерела, можливості, управління, trust |
| VS Code / IDE | `vscode-ide.md` | Extension функції, sidebar, налаштування, сесії |
| SDK | `sdk.md` | Пакети, API, Remote Control |
| Windows | `windows.md` | Все Windows-специфічне ( ТАЖ в основній категорії з тегом [Win] ) |
