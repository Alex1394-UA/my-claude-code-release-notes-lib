# Реліз 2.1.120 — 2026-04-28

## Нові можливості
- PowerShell як заміна Git Bash на Windows — Git Bash більше не потрібен (категорія: windows, core)
- `claude ultrareview [target]` CLI підкоманда для non-interactive `/ultrareview` з CI (категорія: cli)
- `${CLAUDE_EFFORT}` змінна в skill content (категорія: skills-agents)
- `AI_AGENT` змінна середовища для subprocesses (категорія: env-vars)
- Spinner tips приховані коли desktop app/skills вже встановлені (категорія: core)
- Підказка PgUp/PgDn для скролінгу (категорія: core)
- Auto mode denial message з посиланням на docs (категорія: core)
- `claude plugin validate` розширено новими полями (категорія: cli)
- Auto-compact показує `auto` замість оманливого token count (категорія: core)

- Scrollback duplication в non-fullscreen mode (категорія: core)
- `DISABLE_TELEMETRY` / `CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC` не подавляли telemetry для API/enterprise (категорія: env-vars)
- False-positive "Dangerous rm operation" в auto mode (категорія: settings, windows)
- Long selection menus обрізались в fullscreen (категорія: core)
- Write tool output згортався замість розгортання в fullscreen (категорія: core)
- Slash command picker стрибав при наборі (категорія: core)
- `/plugin` marketplace падав при невідомому source format (категорія: plugins)
- `/usage` відкриває нативний dialog в VS Code (категорія: vscode-ide)
- Voice dictation поважає `language` setting в VS Code (категорія: vscode-ide)
- `find` вичерпував file descriptors на великих directory trees (macOS/Linux native) (категорія: core)
