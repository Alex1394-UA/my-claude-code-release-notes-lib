## Виправлення
- Unbounded memory growth (multi-GB) при багатьох зображеннях (категорія: core)
- `/usage` memory leak до ~2GB (категорія: core)
- Memory leak при long-running tools без progress event (категорія: core)
- Bash tool непридатний коли CWD видалено mid-session (категорія: core)
- `--resume` crash при старті в external builds (категорія: core)
- `--resume` відмова на corrupted transcript lines (категорія: core)
- `thinking.type.enabled` помилка на Bedrock inference profile ARNs (категорія: mcp)
- Microsoft 365 MCP OAuth duplicate `prompt` parameter (категорія: mcp)
- Scrollback duplication при Ctrl+L/redraw на tmux, GNOME Terminal, Windows Terminal, Konsole (категорія: core)
- claude.ai connectors silent disappear при transient auth error (категорія: mcp)
- "Always allow" rules не зберігались в remote sessions після restarts (категорія: settings)
- `NO_PROXY` не працював для native build з `managed-settings.json` (категорія: env-vars)
- Managed settings approval prompt завершував сесію (категорія: settings)
- `/usage` stale OAuth token — "/rate limited" (категорія: core)
- Invalid legacy enum values ламали весь settings.json (категорія: settings)
- `/usage` dialog обрізався при вимкненому no-flicker (категорія: core)
- `/focus` показував "Unknown command" без fullscreen renderer (категорія: core)
- Embedded grep/find/rg wrappers падали при видаленому binary (категорія: core)
- Зменшено peak FD usage при `find` в Bash tool (категорія: core)

## Важливі зміни
- Значні memory leak виправлення (~2GB+) для довгих сесій
- PostToolUse хуки тепер можуть замінити output для всіх інструментів, не лише MCP
