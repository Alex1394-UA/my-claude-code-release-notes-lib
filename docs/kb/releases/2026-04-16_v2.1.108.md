# Реліз 2.1.108 — 2026-04-16

## Нові можливості
- `ENABLE_PROMPT_CACHING_1H` env var для 1-годинного prompt cache TTL (категорія: env-vars)
- `FORCE_PROMPT_CACHING_5M` для примусового 5-хв TTL (категорія: env-vars)
- Session recap: контекст при поверненні, `/recap`, configurable через `/config` (категорія: core, skills-agents)
- Вбудовані slash-команди (`/init`, `/review`, `/security-review`) через Skill tool (категорія: skills-agents)
- `/undo` аліас для `/rewind` (категорія: skills-agents)

## Покращення
- `/model` попереджає при switch mid-conversation (категорія: core)
- `/resume` picker за замовч. показує сесії поточної директорії (категорія: cli)
- Error messages: server rate limits vs plan limits, 5xx/529 → status.claude.com (категорія: core)
- Зменшено memory footprint для Read/Edit/syntax highlighting (категорія: core)
- "verbose" індикатор при Ctrl+O (категорія: core)
- Попередження при старті коли prompt caching вимкнено (категорія: core)

## Виправлення
- Paste в `/login` code prompt (регресія з 2.1.105) (core)
- `DISABLE_TELEMETRY` subscribers fallback до 5-хв TTL замість 1 год (env-vars)
- Agent tool permission prompt в auto mode (core)
- Bash tool без output коли `CLAUDE_ENV_FILE` закінчується на `#` comment (env-vars)
- `--resume` втрата кастомного імені та кольору (cli)
- Session titles placeholder при короткому привітанні (core)
- Terminal escape codes після `--teleport` (cli)
- `/feedback` retry не працює без редагування (core)
- `--teleport` та `--resume` precondition errors мовчазний вихід (cli)
- Remote Control session titles перезаписуються (sdk)
- `--resume` truncating при самодостатніх повідомленнях (cli)
