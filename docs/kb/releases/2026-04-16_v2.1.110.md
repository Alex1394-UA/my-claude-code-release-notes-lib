# Реліз 2.1.110 — 2026-04-16

## Нові можливості
- `/tui` команда та `tui` setting — flicker-free rendering (`/tui fullscreen`) (категорія: cli)
- Push notification tool — mobile push від Claude при Remote Control (категорія: core)
- `/focus` команда — окремий toggle focus view (категорія: skills-agents)
- `autoScrollEnabled` config для вимкнення auto-scroll в fullscreen (категорія: settings)
- Опція коментованого контексту в зовнішньому редакторі через `/config` (категорія: settings)

## Покращення
- `Ctrl+O` тепер toggle transcript; focus view окремо через `/focus` (категорія: keybindings)
- `/plugin` Installed tab: favorites зверху, disabled за fold (категорія: plugins)
- `/doctor` попередження при MCP сервері в кількох конфіг-скопах (категорія: core)
- `--resume`/`--continue` resurrects scheduled tasks (категорія: cli)
- `/context`, `/exit`, `/reload-plugins` працюють з Remote Control (категорія: sdk)
- Write tool повідомляє модель про IDE diff edits (категорія: vscode-ide)
- Bash tool застосовує документований максимальний таймаут (категорія: core)
- SDK sessions читають `TRACEPARENT`/`TRACESTATE` (категорія: sdk)
- Session recap увімкнено для telemetry-disabled користувачів (категорія: core)

## Виправлення
- MCP tool calls зависають при розриві з'єднання на SSE/HTTP (mcp)
- Non-streaming fallback багатохвилинні зависання (core)
- Session recap не показується в focus mode (core)
- Високе CPU в fullscreen при виділенні text (core)
- Plugin install не встановлює dependencies з `plugin.json` (plugins)
- Skills з `disable-model-invocation: true` fail при `/<skill>` (skills-agents)
- `--resume` показує перший промпт замість `/rename` імені (cli)
- Повторна появу повідомлень з черги при multi-tool-call (core)
- Session cleanup не видаляє subagent транскрипти (core)
- Dropped keystrokes після CLI relaunch (core)
- Пошкоджений рендеринг старту в macOS Terminal (core)
- "Open in editor" hardened проти command injection (core)
- PermissionRequest хуки не перевіряють `deny` rules повторно (hooks)
- PreToolUse `additionalContext` втрачається при помилці tool call (hooks)
- stdio MCP stray non-JSON lines regression (mcp)
- Headless auto-title зайвий Haiku запит (sdk)
- Надмірне виділення пам'яті при piped Ink output (core)
- `/skills` menu не скролиться в fullscreen (skills-agents)
- Remote Control generic error замість re-login prompt (sdk)
- Remote Control renames з claude.ai не зберігаються локально (sdk)
