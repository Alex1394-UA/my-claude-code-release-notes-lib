# Реліз 2.1.91 — 2026-04-06

## Нові можливості
- MCP `_meta["anthropic/maxResultSizeChars"]` annotation (до 500K) (категорія: mcp)
- `disableSkillShellExecution` setting для вимкнення shell в skills/командах (категорія: settings)
- Підтримка багаторядкових промптів в `claude-cli://open?q=` (%0A) (категорія: cli)
- Плагіни можуть постачати `bin/` executables та викликати як голі команди (категорія: plugins)

## Виправлення
- Transcript chain breaks на `--resume` при async write failures (категорія: cli)
- `cmd+delete` не видаляв до початку рядка (iTerm2, kitty, WezTerm, Ghostty, Windows Terminal) (категорія: core)
- План-режим в remote sessions втрачав план-файл після рестарту контейнера (категорія: core)
- JSON schema валідація для `permissions.defaultMode: "auto"` (категорія: settings)
- Windows version cleanup не захищав rollback копію активної версії (категорія: windows)
- `/feedback` тепер пояснює чому недоступний (категорія: core)
