# Реліз 2.1.86 — 2026-03-28

## Нові можливості

- `X-Claude-Code-Session-Id` заголовок API — агрегація запитів по сесії для проксі
- `.jj` та `.sl` додано до списку виключень VCS-директорій (Jujutsu, Sapling)
- Read tool: компактний формат номерів рядків + дедублікація незмінних перечитувань
- Файли пам'яті тепер підсвічуються при наведенні та відкриваються при кліку
- Описи навичок у `/skills` обмежені 250 символами
- `/skills` меню тепер відсортоване алфавітно
- Auto mode тепер показує "unavailable for your plan" замість "temporarily unavailable"
- Зменшення токенних витрат при `@` згадках файлів

- `--resume` падав на сесіях створених до 2.1.85 (tool_use/tool_result розбіжність)
- Write/Edit/Read не працювали на файлах поза project root з умовними skills/rules
- Зайві запису на диск при кожному виклику skill
-OOM краш при `/feedback` на довгих сесіях
- `--bare` режим викидав MCP інструменти в інтерактивних сесіях
- Скорочення `c` копіювало лише ~20 символів OAuth URL
- Masked input (OAuth code) витікав токен при обгортанні на кілька рядків
- Marketplace plugin скрипти не працювали на macOS/Linux з 2.1.83
- Statusline показував модель іншої сесії при кількох інстансах
- Scroll не слідкував за новими повідомленнями після wheel scroll
- `/plugin uninstall`: `n` тепер правильно видаляє плагін
- Порожній транскрипт після кліку + Enter
- Підказка `ultrathink` не зникала після видалення ключового слова
- Memory growth від render кешів на довгих сесіях
- Startup stalls при багатьох claude.ai MCP connectors (macOS keychain cache 5s → 30s)
- [VSCode] Хибне "Not responding" при довгих операціях
